The OpenRPC Specification is a vendor-neutral, open-source API description. Everyone is welcome to participate in the evolution of the OpenRPC spec.

Get involed on Github

  • Participate on Github:
    • Contribute to close issues on the project board.
    • Join a public planning meeting. (pinned in issues)

Leverage OpenRPC

The OpenRPC specification is free and open source!

    • Find the latest version here.
    • Already using OpenRPC? Tell us about it on the #open-rpc channel on Discord.

Learn More

  • The OpenRPC playground is a safe space to experiment with your very own open-rpc service description documents: https://playground.open-rpc.org/
    • Join our planning meetings and ask questions.
    • Join the discourse on Github and Discord
